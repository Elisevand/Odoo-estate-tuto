# Odoo Estate Tutorial

This repository contains my implementation of the **Estate** module, following the official [Odoo tutorials](https://www.odoo.com/documentation/18.0/developer/tutorials/server_framework_101.html).

## About

The Estate module was developed as part of my learning process with Odoo.
It covers the basics of creating a custom module, including models, views, business logic, and integration with Odoo’s accounting features.

## How to use

- Place the `estate` (and optionally `estate_account`) module in your Odoo addons path.
- Update the app list and install the module from the Odoo interface.
